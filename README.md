# PyStack Academy

A Django-based web application for learning Python Full Stack Development.

## Features
- Home Page
- Courses
- Login/Register
- Admin Panel

## Tech Stack
- Django
- HTML, CSS
- SQLite (will upgrade to PostgreSQL)

## Setup

```bash
git clone <repo-url>
cd pystackacademy
python -m venv venv
pip install -r requirements.txt
python manage.py runserver
